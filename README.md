# QuakeBR

A Minecraft Battle Royale plugin featuring custom weapons, raycast gun mechanics, and match flow management.

## Features
- **Custom Weapons**: AK-47 (Crossbow-based automatic rifle) and Railgun (instant raycast sniper inspired from Quake Minigame).
- **Match Commands**: Commands to join, leave, and start matches.
- **Weapon Testing**: Admin command to give custom items for easy testing.

## Commands
- `/qbr join` - Join the Battle Royale queue.
- `/qbr leave` - Leave the queue.
- `/qbr start` - Start the game (Operators only).
- `/get <ak|railgun>` - Obtain a custom weapon (Operators only).
